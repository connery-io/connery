# Connectors

Below is a manually curated list of public connectors you can install on your runner.<br/>
If you want to add your connector to this list, please open a pull request.

Visit the GitHub repository of the connector to learn more about it, create an issue, or contribute to it.

:::note Helpful resources

- [What is a connector?](/docs/introduction/core-concepts#connector)
- [How to install a connector on the runner?](/docs/getting-started/install-connector-on-the-runner)
- [How to create a connector?](/docs/guides/create-connector)
